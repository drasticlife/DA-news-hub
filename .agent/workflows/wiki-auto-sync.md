새로운 노트를 추가한 후 "노트 추가했으니 WIKI해줘"라고 요청하면 다음 단계를 자동으로 수행합니다.

### 1. 신규 노트 탐색 및 분석
- **대상 경로:** `Moon Life Planner/04. Resources/생활가전제품 WIKI/` 폴더 내부만 탐색합니다.
- 해당 경로에 있는 '무제' 또는 최근 생성된 `.md` 파일을 확인합니다.
- 파일 내용을 분석하여 적절한 하위 카테고리(1~5번 폴더)를 결정합니다.

### 2. 스타일 변환 (설명체 유지 + WIKI 포맷)
- **YAML Frontmatter**: 파일명 기반으로 `title` 추가
- **설명체 유지**: 원본의 친절한 어투를 살리되, 두서없는 부분은 논리적으로 재구성
- **위키 요소 삽입**: 가독성을 위한 콜아웃(`[!abstract]`, `[!info]`), 표(Table), 인포그래픽(Mermaid) 추가

### 3. 파일 이동 및 연결
- 결정된 카테고리 폴더로 이동
- `index.md` 해당 카테고리에 링크 추가
- 기존 관련 노트를 찾아 상호 링크(`[[링크]]`) 생성

### 4. 사이드바 반영 (`DAWikiSidebar.tsx`)
- 해당 카테고리 배열에 새 항목 추가 (`name`, `fileName`, `icon` 자동 지정)

// turbo
### 5. 배포 및 동기화 (Git Push)
- `DA-news-wiki` 폴더에서 변경 사항 스테이징
- "WIKI: [파일명] 추가 및 정리" 메시지로 커밋 후 `origin main`에 푸시
